Política de privacidad

Última actualización: 2026

Aplicación: HarvestSync

Responsable: Sansebas

Contacto: antoniosazon@gmail.com

1. Información que recopilamos

HarvestSync es una aplicación de uso profesional destinada a la gestión de muestras agrícolas. La aplicación puede manejar los siguientes datos:

  • Datos introducidos manualmente por el usuario (valores de muestras, observaciones, etc.).
  • Identificadores de muestras (IDs, boletas, etc.).
  • Imágenes capturadas mediante la cámara del dispositivo.

2. Uso de la cámara

La aplicación utiliza la cámara del dispositivo exclusivamente para capturar imágenes relacionadas con las muestras gestionadas por el usuario.

Las imágenes se utilizan para documentación técnica, control de calidad y generación de informes dentro del sistema.

Las imágenes no se utilizan con fines comerciales ni se comparten con terceros externos al sistema.

3. Almacenamiento de datos

Los datos e imágenes generados por la aplicación se almacenan en servidores privados configurados por la organización.

Estos servidores pueden incluir:

  • Servidores locales
  • Sistemas NAS (almacenamiento en red)
  • Infraestructura interna protegida

4. Funcionamiento offline

La aplicación puede funcionar sin conexión a internet. En estos casos:

  • Los datos se almacenan temporalmente en el dispositivo.
  • Las imágenes se guardan localmente.
  • Cuando se restablece la conexión, los datos se sincronizan automáticamente con el servidor.

5. Conectividad y red privada

Para el acceso a los servidores, la aplicación puede utilizar redes privadas seguras (por ejemplo, VPN como Tailscale).

Esto garantiza que la comunicación entre el dispositivo y el servidor se realice de forma protegida.

6. Compartición de datos

HarvestSync no comparte datos personales ni imágenes con terceros externos.

Toda la información se utiliza exclusivamente dentro del entorno de trabajo de la organización.

7. Seguridad

Se aplican medidas técnicas y organizativas para proteger los datos:

  • Acceso restringido a servidores
  • Redes privadas seguras
  • Control de acceso de usuarios

8. Derechos del usuario

Los usuarios pueden solicitar información sobre los datos gestionados a través del responsable indicado.

9. Cambios en la política

Esta política puede actualizarse para adaptarse a mejoras o cambios en la aplicación.
